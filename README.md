# Co Pilot Guard OS v0.8.0.1 — Overlay, Safe Area & Vertical Containment Repair

A surgical stability release built on v0.8.0 Mission Command Center.

## Repairs
- Guardian and Timeline floating controls automatically yield whenever a capture or incident modal is open.
- Modal overlays now own a defined top interaction layer.
- Desktop mission content scrolls inside the center workspace instead of extending below the viewport.
- Timeline and mission workspace maintain independent vertical scrolling.
- Capture and incident sheets retain protected bottom action space.
- Wide desktop overlays are constrained between the 220px navigation rail and 390px Timeline Engine.
- Tablet and mobile overlays remain viewport-bound and safe-area aware.
- Mission Command workflow, evidence behavior, Timeline Engine, and mobile layout are otherwise unchanged.
