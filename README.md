# Co Pilot Security Marketplace — v1.0.0.3

## Marketplace Layout Foundation

This release rebuilds the agency marketplace around a stable command-center layout system.

### Included
- Dedicated developer preview rail that no longer floats over application content
- Fixed desktop application geometry with reserved header and sidebar regions
- Three-column marketplace command center
- Larger map-first workspace
- Independently scrolling opportunity, operations, and activity panels
- Fixed bottom utility row for guards and quick actions
- Stable heights that do not shift after switching between Agency and Guard portals
- Responsive tablet reflow and mobile stacking
- Live guard pulse and route-glow micro-animations
- Updated build badge and package version

The marketplace behavior remains unchanged: agencies can accept a job, move it into Operations, and continue competing for additional work with remaining capacity.
