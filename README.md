# Co Pilot Guard OS v0.8.2 — Evidence & Incident Workspace

A focused operational release built on the stable v0.8.1 Checkpoint Intelligence foundation.

## What changed
- Unified Mission Evidence workspace grouped by checkpoint.
- Mission-level counts for photos, videos, notes, and incident drafts.
- Evidence cards show checkpoint ownership, GPS verification, and synchronization state.
- Incident reports now support draft, submitted, and resolved states.
- Incident reports include severity, description, people or vehicle details, attached media, exact location, and emergency-services contact state.
- Guards can submit, resolve, or delete incidents from the mission workspace.
- Checkpoint completion is protected while an incident draft remains open.
- Final patrol submission is protected from unresolved incident drafts.
- Evidence and incident state changes continue feeding the Mission Timeline through the existing Mission Engine.

## Preserved foundations
- v0.8.1 Checkpoint Intelligence and completion validation.
- v0.8.0.2 mobile Action Zone spacing.
- Guardian and Timeline overlay safety.
- Desktop Mission Command containment and independent timeline scrolling.
