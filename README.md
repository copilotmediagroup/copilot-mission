# Co Pilot Security Marketplace — v1.0.0.1

## Agency Marketplace Map & Capacity Engine

This release establishes the agency-side marketplace foundation while preserving the complete guard mission engine.

### Included
- Agency Marketplace is the default portal
- Live opportunity map with standard, priority, and emergency jobs
- Agency-owned online guard markers
- Open-job filters and opportunity feed
- Real guard-capacity summary
- Closest-available-guard matching
- Agency acceptance flow
- Accepted jobs leave the open marketplace pool only
- Winning agency retains complete marketplace access
- Accepted jobs enter Operations
- Operations page confirms remaining capacity and links back to Marketplace
- Marketplace and Operations live navigation badges
- Responsive desktop and mobile layouts
- Developer portal switcher between Agency Marketplace and Guard Mission

### Marketplace rule
Accepting work changes agency capacity, not agency access. Agencies can continue finding work for every genuinely available guard.

### Run
```bash
npm install
npm run dev
```
