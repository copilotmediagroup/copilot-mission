# Co Pilot Security Marketplace v1.0.0.2

## Premium Agency Marketplace Experience

This release rebuilds the agency marketplace around the approved premium, map-first visual direction.

### Included
- High-density desktop agency command center
- Live marketplace map with open, priority, emergency, and agency guard markers
- Open-opportunity cards with ETA, distance, property type, duration, price, and accept action
- Persistent agency capacity visualization
- Active operations preview while the agency continues competing
- Available guard roster and quick actions
- Marketplace live activity feed
- Separate Marketplace and Operations workspaces
- Responsive mobile layout designed to prevent card and text overlap
- Job acceptance removes only the accepted job and transfers it to Operations

### Upload
Upload all six top-level repository items directly to the GitHub repository root, then allow Bolt to synchronize the external changes.
