# Co Pilot Guard Living Dashboard v0.1.0

Visual-only React/Vite prototype for the approved Guard Mission Mode experience.

## Run locally

```bash
npm install
npm run dev
```

## Included

- Personal Mode / Offline white dashboard
- Mission Mode dark dashboard
- Eight mission states
- Developer state switcher
- Clickable state rail
- Responsive mobile and desktop presentation
- Animated radar, state transitions, route, patrol checklist, proof, and completion views

No authentication, Supabase, GPS, or backend logic is included in this prototype.
