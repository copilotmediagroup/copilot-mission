# Co Pilot Security Marketplace — v1.0.0.5

## Marketplace UI 2.0

This release rebuilds the Agency Marketplace responsive foundation.

### Desktop
- Stable three-column command-center layout
- Taller bottom utility strip
- Seven-column guard roster with isolated name and distance rows
- No overlapping guard labels

### Mobile
- Independent mobile layout rather than a compressed desktop dashboard
- Full-width header and content
- Marketplace status strip
- Full-width live map
- Stacked opportunity, capacity, operations, activity, guard, and quick-action sections
- Horizontally scrollable guard roster
- Fixed five-item bottom navigation
- Horizontal overflow containment

Accepted jobs continue moving into Operations while the agency remains able to compete for additional marketplace work.


## v1.0.0.5 refinement
- Removed the duplicated mobile top offset that created the blank header gap.
- Placed the KPI strip fully in normal document flow below the sticky app header.
- Tightened the mobile header and KPI cards while preserving readable touch targets.
